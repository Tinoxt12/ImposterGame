# Imposter — Local Pass & Play

Mobile-first, pass-and-play social deception game inspired by "Imposter".

Stack
- Next.js
- React + TypeScript
- TailwindCSS
- Framer Motion

This app is local-only (single device). No backend required.

Quick start

```bash
pnpm install
pnpm dev
```

Deploy
- Deploy to Vercel by pointing the project to this repository (no server).

Gameplay
- Add 3–8 player names, choose mode (Classic or Decoy)
- Start game, pass phone around for private reveals
- Discuss and vote in person, then reveal imposter
- Tap "Next Round" to play again
